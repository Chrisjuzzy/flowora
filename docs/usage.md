# Usage Guide
Flowora — Where AI Agents Flow Together.

## First Run
1. Open the dashboard at `http://localhost`.
2. Register a new user and sign in.
3. Create an agent in Agent Studio.
4. Run the agent and review execution logs.

## Workflows
1. Open Workflow Builder.
2. Create a workflow and connect agent nodes.
3. Run the workflow and view outputs.

## Marketplace
1. Publish an agent with pricing and category.
2. Explore public listings.
3. Install or clone agents to your workspace.

## Public Pages
- Share a published agent at `/a/{agent_id}`.
- View public system agents at `/agents/{slug}`.
- Explore community standings on `/leaderboard`.
- Try the interactive demo at `/demo`.
- Share agent results at `/share/{run_id}`.

## Observability
1. Prometheus: `http://localhost:9090`
2. Grafana: `http://localhost:3001`
3. Review API latency, queue depth, and worker health.
